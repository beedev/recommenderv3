    Observability Architecture - LangSmith Integration¶

    File: src/backend/app/services/observability/langsmith_service.py

    The Observability System provides workflow tracing, performance monitoring, and error tracking through LangSmith integration.

    Overview¶

    The LangSmith observability service enables:

    • Automatic Tracing: @traceable decorator for method instrumentation
    • Performance Metrics: Execution time, token usage, success rates
    • Error Monitoring: Exception tracking with context
    • Agent Action Logging: Track Agent 1, 2, 3 operations
    • Workflow Visualization: End-to-end conversation flow in LangSmith UI

    def __init__(self):
    """Initialize LangSmith client with .env configuration."""
    self.api_key = os.getenv("LANGSMITH_API_KEY")
    self.project = os.getenv("LANGSMITH_PROJECT", "Recommender")
    self.enable_tracing = os.getenv("LANGSMITH_TRACING", "true").lower() == "true"

    # Initialize client
    if self.api_key and self.enable_tracing:
        try:
            self.client = Client(api_key=self.api_key)

            # Set environment variables for @traceable decorator
            os.environ["LANGCHAIN_TRACING_V2"] = "true"
            os.environ["LANGCHAIN_API_KEY"] = self.api_key
            os.environ["LANGCHAIN_PROJECT"] = self.project
            os.environ["LANGCHAIN_ENDPOINT"] = "https://api.smith.langchain.com"

            logger.info(f"LangSmith initialized for project: {self.project}")
            logger.info("✓ LangSmith tracing enabled with environment variables")
        except Exception as e:
            logger.warning(f"LangSmith client initialization failed: {e}")
            self.client = None
    else:
        self.client = None
        logger.info("LangSmith tracing disabled")

    Environment Variables Set: - LANGCHAIN_TRACING_V2=true - Enable LangSmith tracing v2 - LANGCHAIN_API_KEY - API key for authentication - LANGCHAIN_PROJECT - Project name for trace organization - LANGCHAIN_ENDPOINT - LangSmith API endpoint

    @traceable(name="configurator_workflow", run_type="chain")
async def track_workflow_execution(
    self,
    session_id: str,
    user_message: str,
    current_state: str,
    result: Dict[str, Any]
):
    """Track complete workflow execution in LangSmith."""

    workflow_metrics = {
        "session_id": session_id,
        "current_state": current_state,
        "user_message": user_message,
        "ai_response": result.get("ai_response", ""),
        "checkpoint_count": result.get("checkpoint_count", 0),
        "total_messages": len(result.get("messages", [])),
        "had_error": result.get("error") is not None,
        "timestamp": datetime.utcnow().isoformat()
    }

    # Additional agent metrics
    if "agent_actions" in result:
        workflow_metrics["agent_action_count"] = len(result["agent_actions"])

    if "neo4j_queries" in result:
        workflow_metrics["neo4j_query_count"] = len(result["neo4j_queries"])

    if "llm_extractions" in result:
        workflow_metrics["llm_extraction_count"] = len(result["llm_extractions"])

    logger.info(f"LangSmith workflow tracked: {workflow_metrics}")

    @traceable Decorator¶

    Overview¶

    The @traceable decorator automatically instruments methods for LangSmith tracing.

    Import: 

    1
    from langsmith import traceable

    Decorator Parameters¶

    name (required)¶

    Human-readable name for the trace (appears in LangSmith UI).

    Example: 

    1
2
3
    @traceable(name="extract_parameters", run_type="llm")
async def extract_parameters(...):
    pass

    run_type (required)¶

    Type of operation being traced. Common values:

    • "llm": LLM API calls (OpenAI, Anthropic)
    • "chain": Multi-step workflows
    • "retriever": Database queries (Neo4j, PostgreSQL)
    • "tool": Tool/function calls

    Example: 

    1
2
3
4
    @traceable(name="search_products", run_type="retriever")
async def search_products(...):
    # Neo4j query
    pass

    Variable Details¶

    LANGSMITH_API_KEY¶

    Purpose: Authentication for LangSmith API

    How to Get: 1. Sign up at https://smith.langchain.com 2. Navigate to Settings → API Keys 3. Create new API key 4. Copy key (starts with lsv2_pt_)

    Format: lsv2_pt_ + 32-character alphanumeric string

    LANGSMITH_PROJECT¶

    Purpose: Project name for organizing traces

    Default: "Recommender"

    Customization: - Use different project names for dev/staging/production - Example: "Recommender-Dev", "Recommender-Prod"

    LANGSMITH_TRACING¶

    Purpose: Enable/disable tracing

    Values: - true - Enable tracing (default) - false - Disable tracing (no LangSmith API calls)

    Use Cases: - Set to false for local development without LangSmith account - Set to false to reduce API costs - Set to true for production observability

    Graceful Degradation¶

    If LangSmith is not configured:

    1
2
3
    if not self.api_key or not self.enable_tracing:
    self.client = None
    logger.info("LangSmith tracing disabled")

    Behavior: - Application continues running normally - No LangSmith API calls made - @traceable decorators are no-ops (no overhead) - Logging still works via Python logging module

from app.services.observability.langsmith_service import get_langsmith_service

langsmith_service = get_langsmith_service()

# Track agent operation
start_time = time.time()

try:
    # Agent 1: Parameter extraction
    parameters = await parameter_extractor.extract_parameters(
        user_message="I need a 500A MIG welder",
        current_state="power_source_selection"
    )

    duration_ms = int((time.time() - start_time) * 1000)

    # Log successful action
    langsmith_service.log_agent_action(
        action_type="parameter_extractor",
        action_name="extract_parameters",
        input_data={
            "user_message": "I need a 500A MIG welder",
            "current_state": "power_source_selection"
        },
        output_data={"parameters": parameters},
        duration_ms=duration_ms,
        success=True,
        error=None
    )

except Exception as e:
    duration_ms = int((time.time() - start_time) * 1000)

    # Log failed action
    langsmith_service.log_agent_action(
        action_type="parameter_extractor",
        action_name="extract_parameters",
        input_data={
            "user_message": "I need a 500A MIG welder",
            "current_state": "power_source_selection"
        },
        output_data=None,
        duration_ms=duration_ms,
        success=False,
        error=str(e)
    )
