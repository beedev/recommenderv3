    Integrated Cooler Implementation¶

    Date: 2025-11-03 Status: ‚úÖ COMPLETED Feature: PowerSources with built-in coolers automatically skip cooler selection

    Overview¶

    Some PowerSource units have built-in (integrated) coolers. When a user selects such a PowerSource: - ‚úÖ Cooler selection state is automatically skipped - ‚úÖ has_integrated_cooler: true flag is added to PowerSource data - ‚úÖ Flag is passed to search_interconnector() and search_torch() - ‚úÖ Cooler compatibility checks are skipped in product searches

    Key Benefit: Streamlined user experience - users don't see irrelevant cooler selection prompts.

    class ComponentApplicability(BaseModel):
    """
    Component applicability flags for a power source

    Applicability Values:
    - "mandatory": Component is required and cannot be skipped
    - "conditional": Component is optional based on other selections
    - "optional": Component is optional
    - "not_applicable": Component is not applicable for this power source
    - "integrated_cooler": PowerSource has built-in cooler (skip cooler selection)

    Special Case - Integrated Cooler:
    When Cooler = "integrated_cooler", the chatbot will:
    - Skip cooler_selection state entirely
    - Add has_integrated_cooler: True to PowerSource data
    - Pass this flag to search_interconnector() and search_torch()
    - Ensure cooler compatibility checks are skipped in searches
    """

    Formula: 

    1
    has_cooler = bool(cooler_gin) and not integrated_cooler

    Result: | cooler_gin | integrated_cooler | has_cooler | Behavior | |------------|-------------------|------------|----------| | None | False | False | Cooler required ‚Üí WHERE NOT EXISTS cooler | | "0123..." | False | True | Cooler present ‚Üí MATCH cooler relationship | | None | True | False | Cooler NOT required (integrated) | | "0123..." | True | False | Cooler NOT required (integrated, even if selected) |

    When has_cooler = False and integrated_cooler = True: - ‚úÖ Cooler MATCH clause is SKIPPED - ‚úÖ Cooler WHERE NOT EXISTS clause is ADDED (excludes products requiring external cooler) - ‚úÖ Only products compatible with PowerSource (without cooler dependency) are returned
